# Student Grade Tracker

A command-line Python application that reads student academic records from a CSV dataset, processes individual subject averages while gracefully skipping missing grades, calculates comprehensive class-level performance metrics, and exports a polished text summary report.

## Project Structure

```text
starter/
├── data/
│   └── students.csv       # Input file containing student names and raw grades
├── grade_tracker.py       # Main application script containing core logic functions
├── requirements.txt       # Dependency manifest (intentionally empty)
└── README.md              # Setup and usage documentation
